Object detection in this project uses OpenCV DNN with MobileNet-SSD (Caffe).

Put these two files in this folder:
1) MobileNetSSD_deploy.prototxt
2) MobileNetSSD_deploy.caffemodel

Expected paths:
- models/MobileNetSSD_deploy.prototxt
- models/MobileNetSSD_deploy.caffemodel

After that, run the app. Detection will start automatically.
Press D to toggle detection ON/OFF in runtime.
